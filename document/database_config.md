int(11)      |        | 主键，外键，可与第三方联结 || 密码(password)     | varchar (20) |        | 登陆密码                   || 昵称(nickname)     | varchar (20) |        | 昵称                       || 邮箱(email)      | Varchar(10)  | √      | 居住城市                   |### 个人设计表(design)| 字段                 | 类型         | 可为空 | 描述                         || -------------------- | ------------ | ------ |
